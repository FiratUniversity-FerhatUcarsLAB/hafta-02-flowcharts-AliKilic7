🧩 1. Adım Adım Açıklama (Anlaşılır Biçimde)

Kullanıcı Girişi (Authentication):
Kullanıcı sisteme giriş yapar veya yeni bir hesap oluşturur. Giriş bilgileri (e-posta, şifre) doğrulanır.

Ürün Ekleme (Add to Cart):
Kullanıcı beğendiği ürünü seçer ve sepete ekler. Sepette ürünün kimliği (ID), adı, fiyatı ve miktarı tutulur.

Stok Kontrolü (Stock Check):
Ürün sepete eklenmeden önce stokta yeterli miktar olup olmadığı kontrol edilir. Eğer stok yetersizse kullanıcı uyarılır.

İndirim Kodu Uygulama (Discount Code):
Kullanıcı bir indirim kodu girebilir. Kod geçerliyse toplam fiyata indirim uygulanır, geçersizse hata mesajı gösterilir.

Kargo Hesaplama (Shipping Calculation):
Kullanıcının adresine göre kargo ücreti hesaplanır. Belirli bir tutarın üzerindeki siparişlerde ücretsiz kargo uygulanabilir.

Ödeme Aşaması (Payment Processing):
Kullanıcı kredi kartı veya farklı bir yöntemle ödeme yapar. Ödeme başarılı olursa sipariş oluşturulur ve stok güncellenir. Başarısızsa kullanıcıya hata mesajı gösterilir.

🧮 2. Pseudocode (Adım Adım Süreç)
BEGIN

  DISPLAY "Giriş yapınız veya kayıt olunuz."
  user <- LOGIN_OR_REGISTER()
  IF user == INVALID THEN
      DISPLAY "Giriş başarısız."
      EXIT
  ENDIF

  cart <- EMPTY_LIST

  WHILE user wants to add product DO
      product <- SELECT_PRODUCT()
      quantity <- INPUT("Adet: ")

      IF CHECK_STOCK(product, quantity) == TRUE THEN
          ADD_TO_CART(cart, product, quantity)
      ELSE
          DISPLAY "Stok yetersiz!"
      ENDIF
  ENDWHILE

  DISPLAY "İndirim kodunuz var mı?"
  code <- INPUT()
  IF VALIDATE_DISCOUNT(code) == TRUE THEN
      discount <- GET_DISCOUNT_VALUE(code)
  ELSE
      discount <- 0
      DISPLAY "Geçersiz indirim kodu."
  ENDIF

  shipping_cost <- CALCULATE_SHIPPING(user.address, cart.total)
  total_price <- cart.total - discount + shipping_cost

  DISPLAY "Toplam tutar: ", total_price

  payment_status <- PROCESS_PAYMENT(user, total_price)
  IF payment_status == SUCCESS THEN
      UPDATE_STOCK(cart)
      CREATE_ORDER(user, cart, total_price)
      DISPLAY "Ödeme başarılı! Siparişiniz oluşturuldu."
  ELSE
      DISPLAY "Ödeme başarısız! Lütfen tekrar deneyin."
  ENDIF

END
